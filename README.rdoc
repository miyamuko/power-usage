== power-usage - 東京電力の電力供給量をモードラインに表示

* Home URL: https://github.com/miyamuko/power-usage
* Version: 0.0.1


=== SYNOPSIS

1. *power-usage-update-interval* で監視間隔を設定 (デフォルトは 1 時間)
2. *power-usage-mode-line-format* でモードラインをお好みで設定
3. M-x power-usage-mode でモードラインに表示

=== DESCRIPTION

power-usage は http://tepco-usage-api.appspot.com/ から取得した電力消費量を
xyzzy のモードラインに表示します。

=== INSTALL

1. {NetInstaller}[http://www7a.biglobe.ne.jp/~hat/xyzzy/ni.html]
   で power-usage, foo, bar をインストールします。

2. ni-autoload を利用していない場合は、
   ~/.xyzzy または site-lisp/siteinit.l に以下のコードを追加します。

     (require "power-usage")

   ※ ni-autoload を利用している場合は設定は不要です。

3. 設定を反映させるため xyzzy を再起動してください。

   ※siteinit.l に記述した場合には再ダンプが必要です。
